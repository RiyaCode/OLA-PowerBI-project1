# 🚖 OLA Ride Data Analysis – Power BI Dashboard

## 📊 Project Overview

This project presents an **interactive Power BI dashboard** created to analyze OLA ride booking data.
The dashboard provides insights into **booking trends, revenue generation, vehicle usage, ride cancellations, and customer ratings**.

The goal of this project is to transform raw ride data into **meaningful visual insights** using Power BI tools and visualizations.

---

## 🎯 Objectives

* Analyze total ride bookings and successful rides
* Understand revenue generation through different payment methods
* Compare performance of different vehicle types
* Identify major reasons for ride cancellations
* Evaluate customer satisfaction through ratings

---

## 🛠 Tools & Technologies Used

* **Microsoft Power BI**
* **Power Query** (Data Cleaning & Transformation)
* **DAX (Data Analysis Expressions)**
* **Data Visualization Techniques**

---

## 📁 Dataset Information

The dataset contains OLA ride booking data for **July 2024**.

### Important Fields

* `Booking_ID` – Unique identifier for each booking
* `Customer_ID` – Unique identifier for each customer
* `Vehicle_Type` – Type of vehicle booked
* `Booking_Value` – Fare amount of the ride
* `Ride_Distance` – Distance travelled in the ride
* `Payment_Method` – Mode of payment used
* `Booking_Status` – Completed / Cancelled / Incomplete
* `Customer_Rating` – Rating given by customer
* `Driver_Ratings` – Driver performance rating
* `Pickup_Location` – Ride starting point
* `Drop_Location` – Ride destination

---

## 📊 Dashboard Sections

### 1️⃣ Overall Dashboard

Provides a summary of ride booking performance including:

* Total bookings
* Successful bookings
* Average ride distance
* Total ride distance travelled

---

### 2️⃣ Vehicle Type Analysis

Analyzes ride statistics based on vehicle categories such as:

* Prime Sedan
* Prime SUV
* Prime Plus
* Mini
* Auto
* Bike
* E-Bike

Metrics shown:

* Total booking value
* Successful booking value
* Average distance travelled
* Total distance travelled

---

### 3️⃣ Revenue Analysis

Displays revenue insights including:

* Revenue by payment method
* Daily ride revenue trends
* Customer payment preferences

Payment methods analyzed:

* Cash
* UPI
* Credit Card
* Debit Card

---

### 4️⃣ Cancellation Analysis

Shows the number of cancelled rides and reasons behind cancellations.

Common cancellation reasons include:

* Driver not moving
* Driver asked to cancel
* Change of plans
* AC not working
* Wrong address

---

### 5️⃣ Ratings Dashboard

Displays customer ratings for different vehicle types to evaluate service quality.

Vehicle types analyzed:

* Prime Sedan
* Prime SUV
* Prime Plus
* Mini
* Auto
* Bike
* E-Bike

---

## 📈 Key Insights

* Prime category vehicles generate the highest bookings.
* UPI and Cash are the most commonly used payment methods.
* Some cancellations occur due to driver issues or incorrect addresses.
* Customer ratings help evaluate the overall ride experience.
